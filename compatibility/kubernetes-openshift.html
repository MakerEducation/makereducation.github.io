Artificial Intelligence solver for Kubernetes and OpenShift

OptaPlanner runs great on Kubernetes, especially on OpenShift.

Start a Developer Sandbox for OpenShift to deploy one of the OptaPlanner Quickstarts to it. Every use-cases quickstart has README instructions on how to easily deploy it on OpenShift.
